keybindings cheatsheet

Neovim Keybindings

Editing, cut & paste, search & replace, buffers, LSP and screen movement. # stands for a count prefix.

Editing replace, change, undo
r
replace single character
R
enter replace mode
J
join line below to current one with one space in between them
gJ
join line below to current one with no spaces
cc
change entire line
C
change to the end of line
ciw
change inner word
s
delete character and substitute text
S
delete line and substitute text
u
undo
Ctrl r
redo
.
repeat last command
Cut & Paste yank, put, delete
yyY
yank single line
yw
yank from cursor to start of next word
y#m
yank # of lines with m movement (hjkl…)
"ayy
yank current line into named buffer a
p
put (paste) after the cursor
P
put (paste) before the cursor
"aP
put text from buffer a before cursor
dd
delete current line
#dm
delete # of lines with movement (hjkl)
D
delete to end of line
dw
delete until word
x
delete character at cursor
X
delete backwards from cursor
Search & Replace
/pattern
search for pattern in current buffer
?pattern
search for pattern backwards in current buffer
<leader>/
search for pattern backwards in current buffer
:s/old/new
replace next old with new
:s/old/new/g
replace instances of old with new in current line
:%s/old/new/g
replace all old with new throughout file
:%s/old/new/gc
replace all old with new in file with confirmations
Screen Movement scroll & reposition
C-d
move forward half a screen
C-u
move backward half a screen
zz
cursor line to center of screen
zt
cursor line to top of screen
zb
cursor line to bottom of screen
Buffers <leader>b?
S-h
switch to left buffer
S-l
switch to right buffer
<leader>bd
delete buffer
<leader>`
change to previous buffer
<leader>bj
pick buffer
<leader>bp
pin buffer
<leader>.
buffer snacks
<leader>be
buffer explorer

gr
references

gy

K
hover
gK
signature help
ca

cr
code rename
[[
next reference
]]
previous reference

workspace symbols
<leadersS>
document symbolos
