Tmux Keybindings

Personal config — prefix remapped to Ctrl + Space.

Panes
Prefix v
split pane beside (vertical)
Prefix h
split pane below (horizontal)
Prefix x
kill pane
Prefix z
toggle pane zoom (fullscreen)
Ctrl Alt Arrow
move between panes
Ctrl Alt Shift Arrow
resize panes
Windows
Prefix c
new window
Prefix k
kill window
Prefix r
rename window
Alt 1…9
go to specific window
Alt Left/Right
move between windows
Session
Prefix C
new session
Prefix K
kill session
Prefix R
rename session
Prefix NPrefix P
next / previous session
Alt Up/Down
move between sessions
Prefix s
list sessions
Prefix d
detach from session

Prefix [

v

y

Prefix q
reload config
