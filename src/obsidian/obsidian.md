## Files & Notes

| Keys      | Action                |
| --------- | --------------------- |
| `Cmd N`   | create new note       |
| `Cmd O`   | quick switcher        |
| `Cmd W`   | close current tab     |
| `Cmd S`   | save file             |
| `Cmd Shift D` | open today's daily note (custom) |

## Formatting

| Keys          | Action                        |
| ------------- | ------------------------------ |
| `Cmd B`       | bold                          |
| `Cmd I`       | italic                        |
| `Cmd K`       | insert/edit link              |
| `Cmd Shift X` | toggle strikethrough          |
| `Cmd Shift H` | toggle highlight              |
| `Cmd E`       | toggle edit/reading view      |

## Search

| Keys      | Action                |
| --------- | --------------------- |
| `Cmd F`   | search current file   |
| `Cmd Shift F` | search all files   |
| `Cmd H`   | find & replace        |

## Lists (indent & task syntax)

| Keys        | Action            |
| ----------- | ----------------- |
| `Tab`       | indent list item  |
| `Shift Tab` | outdent list item |
| `- `        | bullet list       |
| `1. `       | numbered list     |
| `- [ ]`     | unchecked task    |
| `- [x]`     | checked task      |

## Navigation & Panes

| Keys              | Action                              |
| ----------------- | ------------------------------------ |
| `Cmd P`           | command palette                      |
| `Cmd ,`           | open settings                        |
| `Cmd Alt Left`    | navigate back                        |
| `Cmd Alt Right`   | navigate forward                     |
| `Cmd G`           | open graph view                      |
| `Ctrl Tab`        | next tab (uses ctrl, not cmd)        |
| `Ctrl Shift Tab`  | previous tab                         |
| `Cmd Click`       | open link in new pane                |
| `Cmd Alt Enter`   | open link in new pane (keyboard)     |
| `Cmd Alt V`       | toggle default new-pane split direction |
| `Cmd Shift E`     | reveal active file in navigation (custom) |

## Markdown — Formatting

| Keys        | Action           |
| ----------- | ---------------- |
| `# `        | heading 1        |
| `## `       | heading 2        |
| `### `      | heading 3        |
| `**bold**`  | bold             |
| `*italic*`  | italic           |
| `~~strike~~` | strikethrough   |
| `==mark==`  | highlight        |
| `` `code` `` | inline code     |
| ` ``` `     | code block       |
| `> text`    | blockquote       |
| `---`       | horizontal rule  |

## Markdown — Links & Extras

| Keys                   | Action                       |
| ---------------------- | ---------------------------- |
| `[text](url)`          | link                         |
| `![alt](path)`         | image embed                  |
| `[[Note]]`             | internal link                |
| `[[Note\|alias]]`      | link with display text       |
| `![[Note]]`            | embed note                   |
| `![[Note#Heading]]`    | embed section                |
| `#tag`                 | tag                          |
| `%%comment%%`          | hidden comment               |
| `> [!note]`            | callout                      |
| `^block-id`            | mark block for reference     |
| `[[Note#^block-id]]`   | link to a block              |
